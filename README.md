# Color Clock

A simple React clock built with Vite. It shows the current date above the time and updates every second. The clock is displayed inside a white card centered on a gradient background.


## Features

- Shows date above the time with counting seconds
- White card with border and shadow
- Gradient background
- Uses Helvetica font
- Built with React and date-fns


## Installation

1. Install dependencies:

```bash
npm install
npm install date-fns@2.30.0
```

2. Start the development server:

```bash
npm run dev
```

3. Open the link in your browser to see the clock.


## Project Files

src/App.jsx → main React component
src/App.css → styles for card and clock
src/main.jsx → React entry point
