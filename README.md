Color Clock

This is a simple React clock built with Vite. It shows the current date above the time and updates every second. The clock is inside a white card centered on a gradient background.


// Features

- Shows date and the time with counting seconds
- White card with border and shadow
- Gradient background
- Uses Helvetica font
- Built with React and date-fns


// How to Run

1. Install dependencies:

```bash
npm install
npm install date-fns@2.30.0

2. Start the development server:

npm run dev

3. Open the link in your browser to see the clock.
